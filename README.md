## Project Overview

The Pomodoro Productivity Application is designed to help users improve focus, organization, and productivity through an easy-to-use and visually clean interface. The application combines a Pomodoro timer system with note-taking, checklist management, and productivity tracking features in a single dashboard. The design focuses on simplicity, accessibility, and minimal distractions so users can concentrate on completing work sessions effectively.

## Scope

Develop a Pomodoro Productivity App where users can:
+ Create and manage timed Pomodoro focus sessions
+ Take notes during or after work sessions
+ Create and complete task checklists
+ Track completed focus sessions on a calendar
+ Save productivity data and preferences for later use

## Executing

Download FocusFlow.jar

Open folder/directory in terminal

Run with ``` java -jar FocusFlow.jar ```
